## 智動旅程 For 2022 NKUSTCSIE Hackthon

## Dev. Motivation
- 身處台灣，整合景點方便。
- 因為懶得排行程，想要有個機器人幫我們排xD
- 覺得自動排程系統能夠讓我們充滿驚喜。
